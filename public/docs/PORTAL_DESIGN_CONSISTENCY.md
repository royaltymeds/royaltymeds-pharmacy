# Portal Design Consistency Guide\n\n## Overview\n\nAll three portals (Patient, Doctor, Admin) follow the same responsive design patterns and components while maintaining portal-specific color theming. This ensures a cohesive user experience across the entire application.\n\n---\n\n## Color Schemes by Portal\n\n### Patient Portal - Green Theme\n```\nPrimary: bg-green-600 / text-green-600\nLighter: bg-green-50 / text-green-700\nAccent: border-green-600 / border-green-500\nHover: hover:bg-green-700 / hover:text-green-700\nBadge: bg-green-100 / text-green-800\n```\n\n### Doctor Portal - Blue Theme\n```\nPrimary: bg-blue-600 / text-blue-600\nLighter: bg-blue-50 / text-blue-700\nAccent: border-blue-600 / border-blue-500\nHover: hover:bg-blue-700 / hover:text-blue-700\nBadge: bg-blue-100 / text-blue-800\n```\n\n### Admin Portal - Green Theme\n```\nPrimary: bg-green-600 / text-green-600\nLighter: bg-green-50 / text-green-700\nAccent: border-green-600 / border-green-500\nHover: hover:bg-green-700 / hover:text-green-700\nBadge: bg-green-100 / text-green-800\n```\n\n---\n\n## Consistent Components Across Portals\n\n### 1. Navigation\n\n#### Desktop Navigation (All Portals)\n- Located in layout.tsx\n- Navbar with logo/title\n- Navigation links aligned left\n- User email + logout button right\n- Hidden on mobile (lg:hidden)\n- Uses portal-specific color\n\n```\nPortal      Color               Links\n---         -----               -----\nPatient     Green (bg-green-600) Dashboard, Orders, Prescriptions, Refills, Messages\nDoctor      Blue (bg-blue-600)  Dashboard, Submit Rx, Prescriptions, Patients\nAdmin       Green (bg-green-600) Dashboard, Prescriptions, Orders, Refills, Doctors, Users\n```\n\n#### Mobile Navigation (All Portals)\n- MobileSidebar component\n- Hamburger menu icon (Menu → X toggle)\n- Slide-in sidebar from right\n- Semi-transparent overlay\n- Same links as desktop\n- Visible only on sm-md (hidden lg:hidden)\n- Portal-specific color\n\n### 2. Page Headers\n\nAll page headers follow identical pattern:\n```tsx\n<div className=\"bg-white rounded-lg shadow-sm p-3 sm:p-4 md:p-6 border-l-4 border-[COLOR]-600\">\n  <div className=\"flex flex-col sm:flex-row items-start sm:items-center justify-between gap-2 sm:gap-3 md:gap-4\">\n    <div className=\"min-w-0 flex-1\">\n      <h1 className=\"text-lg sm:text-2xl md:text-3xl font-bold text-gray-900\">\n        Page Title\n      </h1>\n      <p className=\"text-xs sm:text-sm md:text-base text-gray-600 mt-1\">\n        Subtitle\n      </p>\n    </div>\n    <Link href=\"/\" className=\"flex-shrink-0 text-[COLOR]-600 hover:text-[COLOR]-700 font-medium text-xs sm:text-sm whitespace-nowrap\">\n      ← Home/Back\n    </Link>\n  </div>\n</div>\n```\n\n**Features:**\n- Responsive padding: p-3 sm:p-4 md:p-6\n- Colored left border (matches portal theme)\n- Responsive typography\n- Flex layout that stacks on mobile\n- Navigation link with theme color\n\n### 3. Statistics Cards\n\nConsistent across dashboard pages (Patient Home, Doctor Dashboard, Admin Dashboard):\n```tsx\n<div className=\"grid grid-cols-2 md:grid-cols-2 lg:grid-cols-4 gap-2 sm:gap-3 md:gap-4\">\n  <div className=\"bg-white rounded-lg shadow p-3 sm:p-4 md:p-6 border-t-4 border-[COLOR]-600\">\n    <div className=\"flex items-start justify-between gap-2\">\n      <div>\n        <p className=\"text-gray-500 text-xs md:text-sm font-medium\">Label</p>\n        <p className=\"text-lg sm:text-2xl md:text-3xl font-bold text-gray-900 mt-1 sm:mt-2\">\n          {value}\n        </p>\n      </div>\n      <Icon className=\"h-5 w-5 sm:h-6 sm:w-6 md:h-8 md:w-8 text-[COLOR]-600 flex-shrink-0\" />\n    </div>\n  </div>\n</div>\n```\n\n**Features:**\n- 2-column on mobile\n- 2-column on tablet\n- 4-column on desktop\n- Responsive padding and gaps\n- Portal-colored top border and icons\n- Responsive icon sizing\n\n### 4. Content Cards / List Items\n\nConsistent pattern for messages, orders, refills, prescriptions:\n```tsx\n<div className=\"space-y-2 sm:space-y-3 md:space-y-4\">\n  <div className=\"bg-white rounded-lg shadow p-3 sm:p-4 md:p-6 border-t-4 border-[COLOR]-500 hover:shadow-md transition\">\n    <div className=\"flex items-start justify-between gap-2 sm:gap-3\">\n      <div className=\"flex-1 min-w-0\">\n        <h3 className=\"font-semibold text-gray-900 text-sm sm:text-base\">\n          {title}\n        </h3>\n        <p className=\"text-xs sm:text-sm text-gray-600 mt-1\">\n          {content}\n        </p>\n      </div>\n      <span className=\"px-2 sm:px-3 py-1 rounded-full text-xs font-medium whitespace-nowrap flex-shrink-0 bg-[COLOR]-100 text-[COLOR]-800\">\n        {status}\n      </span>\n    </div>\n  </div>\n</div>\n```\n\n**Features:**\n- Responsive spacing\n- Colored border (top, matches theme)\n- Responsive padding\n- Status badges with theme color\n- Hover effect (shadow increase)\n- Touch-friendly gaps and sizing\n\n### 5. Buttons\n\nConsistent button styling:\n```tsx\n<button className=\"px-3 sm:px-4 md:px-6 py-2 sm:py-2.5 md:py-3 bg-[COLOR]-600 text-white rounded-lg hover:bg-[COLOR]-700 transition font-medium text-xs sm:text-sm md:text-base\">\n  Action Text\n</button>\n```\n\n**Features:**\n- Responsive padding\n- Responsive text size\n- Portal theme color\n- Hover state (darker shade)\n- Smooth transition\n\n### 6. Forms\n\nConsistent form input styling:\n```tsx\n<input \n  type=\"text\"\n  className=\"w-full px-3 sm:px-4 md:px-5 py-2 sm:py-2.5 md:py-3 border border-gray-300 rounded-lg text-xs sm:text-sm md:text-base focus:outline-none focus:border-[COLOR]-500 focus:ring-1 focus:ring-[COLOR]-500\"\n  placeholder=\"Input placeholder\"\n/>\n```\n\n**Features:**\n- Full width\n- Responsive padding\n- Responsive text size\n- Colored focus state (matches theme)\n- Accessible focus ring\n\n### 7. Tables (Admin)\n\nConsistent table styling:\n```tsx\n<table className=\"w-full text-sm\">\n  <thead className=\"bg-gray-50 border-b border-gray-200\">\n    <tr>\n      <th className=\"px-3 sm:px-6 py-3 text-left text-xs font-medium text-gray-700 uppercase tracking-wider\">\n        Column Header\n      </th>\n    </tr>\n  </thead>\n  <tbody className=\"divide-y divide-gray-200\">\n    <tr className=\"hover:bg-gray-50 transition\">\n      <td className=\"px-3 sm:px-6 py-3 text-xs sm:text-sm text-gray-900\">\n        Cell content\n      </td>\n    </tr>\n  </tbody>\n</table>\n```\n\n**Features:**\n- Responsive cell padding\n- Hidden columns on mobile (hidden sm:table-cell)\n- Responsive text sizes\n- Hover states\n- Proper spacing\n\n---\n\n## Responsive Breakpoints (Consistent)\n\n| Breakpoint | Width  | Usage |\n|------------|--------|-------|\n| xs | 0px | Mobile phones (default) |\n| sm | 640px | Tablets portrait |\n| md | 768px | Tablets landscape |\n| lg | 1024px | Desktop (desktop nav shows, mobile sidebar hidden) |\n| xl | 1280px | Large desktop |\n| 2xl | 1536px | Extra large screens |\n\n**Grid Behavior:**\n- Mobile (0-640px): 2-column grids, full-width containers\n- Tablet (640-1024px): 2-column grids or 2-col layout\n- Desktop (1024px+): 4-column grids, desktop navigation\n\n---\n\n## Spacing Scale (Consistent)\n\n```\nPadding:    p-3 sm:p-4 md:p-6    (cards, containers)\nGaps:       gap-2 sm:gap-3 md:gap-4  (between items)\nVertical:   space-y-3 sm:space-y-4 md:space-y-6  (sections)\nMargins:    mt-2/3/4, mb-2/3/4, mx-2/3/4\n```\n\n---\n\n## Typography Scale (Consistent)\n\n```\nPage Titles (H1):     text-lg sm:text-2xl md:text-3xl\nSection Titles (H2):  text-base sm:text-lg md:text-2xl\nSubtitles (H3):       text-sm sm:text-base md:text-lg\nBody Text:            text-xs sm:text-sm md:text-base\nSmall Text:           text-xs (no scaling)\nButton Text:          text-xs sm:text-sm md:text-base\n```\n\n---\n\n## Icon Sizing Scale (Consistent)\n\n```\nSmall (inline):       h-5 w-5 sm:h-6 sm:w-6 md:h-8 md:w-8\nMedium (stats):       h-6 w-6 sm:h-8 sm:w-8 md:h-10 md:w-10\nLarge (heroes):       h-8 w-8 md:h-10 md:w-10\nExtra Large:          h-10 w-10 md:h-12 md:h-12\nAlways include:       flex-shrink-0 (prevents overflow)\n```\n\n---\n\n## Mobile Sidebar Base styles (xs)\n2. Small breakpoint (sm:)\n3. Medium breakpoint (md:)\n4. Large breakpoint (lg:)\n5. jumbled order */\nmd:p-6 p-3 sm:p-4\n```\n\n---\n\n## Common Elements Comparison\n\n| Element | Patient | Doctor | Admin | Style |\n|---------|---------|--------|-------|-------|\n| Primary Color | Ensure responsive classes are present\n\n**When modifying existing pages:**\n1. Maintain responsive class structure\n2. Don't remove breakpoint prefixes\n3. Test all breakpoints\n4.
